## 基本数学运算
### 递归方程
变量替换法求解
T(n)=2T(√n)+logn
令 m=logn，则n=2^m， T(2^m)=2T(2^m/2)+m
令 S(m)=T(2^m)，则 S(m/2)=T(2^m/2)
于是 S(m)=2S(m/2)+m
显然 S(m)=Θ(mlogm)
则 T(n)=Θ(lognlog(logn))
1.MASTER定理求解
求解 T(n)=a∙T(n/b)+f(n) 型递归方程，其中a≥1，b>1 是常数，f(n) 是正函数
记住三种情况，可快速求解
其实就是f（n）和n^logb(a)谁大是谁
